# OpenRA2

红色警戒 2 · 共和国之辉 —— 像素级程序生成素材复刻版。全部美术（地形/建筑/单位/特效）与音频（音效/BGM）均由代码程序化生成，无任何外部资源文件。

## 技术栈

- **C++23**（MSVC 2022 BuildTools）
- **raylib 5.x**（渲染/音频，CMake FetchContent 自动拉取）
- **CMake ≥ 3.24**

## 构建与运行

```powershell
cmake -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release
.\build\Release\ra2.exe
```

冒烟测试（无头跑 N 帧逻辑 + 输出各玩家发展诊断 + 全要素强制验证）：

```powershell
.\build\Release\ra2.exe --smoke 20000
```

## 游戏模式

- **遭遇战**：1~3 个 AI，可选阵营（盟军/苏军/中国）、颜色、初始资金、地图尺寸、地图类型
- **战役模式**：3 个任务（边境冲突 / 近海防御 / 决胜时刻），固定种子地图 + 波次增援脚本 + 胜利目标（歼灭 / 坚守）

### 地图类型

| 类型 | 说明 |
|------|------|
| 大陆 | FBM 噪声散布湖区（默认） |
| 岛屿 | 每出生点一座主岛 + 中央争夺岛，其余为海 |
| 湖泊 | 中央大湖，四周环陆地 |

## 操作

| 按键 | 功能 |
|------|------|
| 左键 / 框选 | 选择单位；双击基地车展开 |
| 右键 | 移动 / 攻击 / 采矿 / 登船（按住 A = 攻击移动） |
| S / X / G / T | 停止 / 散布 / 警戒 / 选同类 |
| D | 基地车展开 · 辐射工兵部署辐射区 |
| U | 运输船卸载乘员 |
| Ctrl+0~9 / 0~9 | 编队设定 / 选取 |
| R / Delete | 设集结点 / 卖建筑 |
| H / P / M | 回基地视角 / 暂停 / 音乐开关 |
| +/- / 方向键 | 游戏速度 / 卷屏 |

## 已实现内容

- **三阵营**：盟军 / 苏军 / 中国，差异化单位与建筑树
- **完整经济**：矿脉分布、采矿车自动采矿/卸载、电力系统（低电减速生产）
- **海军全套**：海军船厂、台风潜艇、海蝎、两栖运输船（载员登船/卸载）、水域寻路域（陆/水/两栖）
- **超级武器**：战术核弹（飞行+落点伤害）、闪电风暴（区域随机雷击）、铁幕（载具无敌），建造即向敌方广播侦测警告
- **特殊单位机制**：超时空军团兵（传送移动/抹除攻击）、幻影坦克（静止伪装）、辐射工兵（部署辐射区）、V3 火箭（远程溅射）、磁爆步兵（链式伤害）、狙击手（步兵秒杀）
- **中立科技建筑**：油井（持续资金）、医院（步兵治疗）、机械店（载具维修），工程师占领生效
- **遭遇战 AI**：建造序列（含雷达后海军船厂、失败自动放弃海军）、经济优先级（攒高科/超武）、编队进攻、超武释放
- **战场体验**：EVA 事件播报（中文字幕）、程序合成 BGM（M 开关）、编队/散布/警戒、小地图、战争迷雾
- **自动化验证**：`--smoke` 覆盖经济发展、AI 建造、海军生产、超武释放、战役波次

```
  gfx/     程序生成精灵（地形/建筑/单位/特效）
  sfx/     程序合成音效与 BGM
```
